Game Swap

   Trade your Video Games

Role(s): Full-Stack Web Developer / Designer

Share the joy of gaming. Game Swap lets you register and trade your cartrige games.

Angular was used for our trading system which includes the ability to offer, accept, and deny trades as well as manage which games in your collection are up for trade.

We developed this project using Test-Driven Development enabled by Chai.

BREW

   A Magic: The Gathering Compendium

Role(s): Full-Stack Web Developer / Designer

Description: Brew is a web-application that allows the user to vizualize Magic the Gathering cards in unique and powerful ways.

Add exactly the sets you want to filter from, and then sort the cards by Color, Casting Cost, or Card Type.

Brew employs Amazon Web Services to download and host the images on my own private bucket, and then updates the database to include an image url.

TOOR

   Crowdsource your vacation

Role(s): Front-End Developer & Designer

Ever wanted to go on a vacation, but were unsure as to exactly where or what the best things to there were?

Toor solves that problem in the most powerful way possible, by asking your friends. Built using both the Twitter and the Sabre API's in order to offer destination options, and read suggestions your friends are tweeting at you.

1st place Winner of the SABRE Api award for the Launch 2015 hackathon.
